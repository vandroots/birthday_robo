
Lembrar do aniversário de todo mundo não é uma tarefa fácil. 
Porém, se tiver uma lista contendo o nome e a data de aniversário, é 
possível automatizar esta tarefa! Para reforçar os conceitos de função aprendidos 
desenvolva o seguinte algoritmo:

“Desenvolver um algoritmo que contenha uma lista com o nome e a 
    data de nascimento dos funcionários de uma empresa. Cada vez que for executado, 
    o algoritmo deve enviar um email com uma mensagem que pode ser 
    padrão ou personalizada para todos que estão fazendo aniversário”
